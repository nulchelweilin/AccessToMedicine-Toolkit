# Roadmap for v2.0 - Priority Areas

This document outlines planned enhancements for the next version of the Access-to-Medicine Strategy Toolkit.

## Task List

- [ ] Develop a stakeholder mapping template.
- [ ] Write a guide on technology transfer partnerships.
- [ ] Source and write the first case study.
- [ ] Create a monitoring & evaluation framework template.
- [ ] Add guidance on engaging with local manufacturers.
- [ ] Include a section on digital health technologies for supply chain.
- [ ] Expand tiered pricing guide with more regional case studies.
- [ ] Develop a voluntary licensing agreement template.
- [ ] Create a guide on sustainable procurement practices.
- [ ] Integrate feedback from low-resource settings.

## How to Contribute

Feel free to open an issue or submit a pull request if you would like to work on any of these items or propose new ones.

## Version History

- **v1.0** (Current): Initial release with Tiered Pricing Guide.
- **v2.0** (Planned): Expanded templates and case studies.
